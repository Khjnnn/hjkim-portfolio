# KBO 2026 캘린더 - 야구 경기일정 & 예매일정 모바일 달력

2026 KBO 프로야구 경기일정과 구단별 티켓 예매 오픈일을 한눈에 확인할 수 있는 모바일 최적화 캘린더 웹앱입니다.

## 🎯 프로젝트 개요
- **목적**: KBO 경기일정 확인 및 구단별 예매 오픈일 자동 계산·표시
- **플랫폼**: 웹 기반 모바일 최적화 캘린더 (반응형)
- **배포**: Cloudflare Pages

## 🔗 링크 URL
[https://kbocalendar2026.pages.dev/](https://kbocalendar2026.pages.dev/)

## ⚡ 주요 기능
- 월별 달력 그리드에 경기/예매 마커 표시 (이모지 기반)
- 구단별 티켓 선예매·일반예매 오픈일 자동 계산
- 시리즈 전체 예매 팀(삼성, 롯데) 특수 규칙 지원
- 10개 구단 필터링 (복수 선택 가능, 컬러 칩 UI)
- 경기일정 탭 / 예매일정 탭 전환
- 경기 카드 클릭 시 해당 경기의 예매일 정보 모달 표시
- 공휴일 붉은색 표시 및 공휴일명 라벨 표기
- 금·토·일 시리즈 및 공휴일 경기 하이라이트 (노란 배경)
- Google Analytics 연동 및 쿠팡 파트너스·애드센스 광고 통합

## 🛠️ 기술 스택
- **Frontend**: Vanilla JavaScript (ES Modules), HTML5, CSS3
- **Build Tool**: Vite 8
- **Data**: 월별 경기일정 JSON, 구단별 예매규칙 JSON
- **Design**: CSS Custom Properties, 모바일 퍼스트 반응형
- **Font**: Gmarket Sans
- **Hosting**: Cloudflare Pages (Git 연동 자동 배포)

## 🚀 개발 과정
1. KBO 공식 일정 기반 월별 경기 JSON 데이터 생성 (3~8월)
2. 달력 그리드 렌더링 모듈 구현 (월 이동, 날짜 선택, 마커)
3. 구단별 예매규칙 JSON 설계 및 선예매·일반예매 오픈일 자동 계산 로직
4. 시리즈 전체 예매 팀(삼성, 롯데) 특수 그룹핑 로직 구현
5. 10개 구단 컬러 칩 필터 UI 및 필터링 로직 적용
6. 경기 카드 클릭 → 예매정보 모달 (하단 슬라이드업) 구현
7. 공휴일 표시, 주말 하이라이트, 광고 통합 등 UX 개선
8. Cloudflare Pages 배포 및 Google Analytics 연동

## 💡 학습한 점
- Vanilla JS 모듈 시스템(ES Modules)을 활용한 관심사 분리 아키텍처
- CSS Custom Properties 기반 일관된 테마 시스템 설계
- 날짜 계산 로직에서 시리즈 그룹핑 및 요일 기반 오프셋 처리
- 이벤트 위임(Event Delegation) 패턴으로 동적 요소 클릭 처리
- Vite를 활용한 빠른 개발 환경 구축 및 정적 사이트 빌드
- Cloudflare Pages Git 연동 자동 배포 파이프라인 구성

## 📱 특징
- 모바일 퍼스트 설계로 최대 500px 컨테이너 내 최적화된 레이아웃
- 구단별 고유 컬러(primary, light, text) 매핑으로 시각적 구분
- 탭 전환 시 마커·상세 패널 즉시 갱신 (캐시 활용)
- 하단 슬라이드업 모달로 네이티브 앱 느낌의 UX 제공
- 공휴일명 자동 표시 및 붉은색 강조로 직관적 달력 경험
