# 🎫 티베이 티켓 데이터 분석 대시보드

티베이 API 크롤링을 통한 티켓 판매 현황 분석 및 BI 대시보드 개발 프로젝트

## 🎯 프로젝트 개요
- **목적**: 티베이 티켓 판매 데이터를 실시간으로 수집 및 분석하여 비즈니스 인사이트 제공
- **플랫폼**: 웹 기반 BI 대시보드
- **배포**: 내부 서버

## 🔗 링크 URL
[개발 중] - 내부 서버에서 테스트 진행 중

## ⚡ 주요 기능
- 티켓베이 자동 크롤링 및 데이터 수집
- DB를 통한 실시간 데이터 저장 및 관리
- 티켓 판매 현황 시각화 대시보드
- 카테고리별 판매 동향 분석
- 실시간 트래픽 및 인기 티켓 순위
- 데이터 기반 비즈니스 의사결정 지원

## 🛠️ 기술 스택
- **Backend**: Node.js, Express, Python (크롤링)
- **Database**: DB (SQLite 기반)
- **Frontend**: HTML/CSS/JavaScript, Chart.js
- **API**: 티켓베이 웹 크롤링
- **데이터 처리**: Cheerio, Axios
- **배포**: 내부 Linux 서버

## 🚀 개발 과정
1. 티켓베이 API 분석 및 크롤링 전략 수립
2. 데이터 수집 자동화 스케줄러 구현
3. DB 스키마 설계 및 데이터 저장 구축
4. 실시간 데이터 파이프라인 개발
5. 대시보드 UI/UX 설계 및 구현
6. 데이터 시각화 라이브러리 연동
7. 성능 최적화 및 안정화 작업
